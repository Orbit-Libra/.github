## 프로젝트 소개
대학도서관 데이터와 예측 모델을 통합한 LLM/에이전트 마이크로서비스 아키텍처입니다.
- 사용자 서비스, 데이터 서비스, LLM 서비스, 에이전트 서비스로 구성
- Oracle XE 기반 RDB + Chroma 기반 벡터 스토어
- LangGraph로 멀티턴 오케스트레이션
