## 프로젝트 소개
대학도서관 데이터를 통한 대학교 별 학습환경 예측 및 대화형 채팅 봇 서비스
- 각 대학교의 학습환경 점수 예측 및 미래점수 예측
- 유저 데이터 기반 학습도 예측 및 소속대학 & 유사점수 대학 비교 분석
- LangGraph 및 MCP를 활용한 AI 에이전트 구현
